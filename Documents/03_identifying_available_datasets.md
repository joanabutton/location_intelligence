# 1. Possible data sources

## INE - Instituto Nacional de Estatistica

Look for:

- Resident population
- Households
- Population density
- Age structure
- Families with children

Ideally at:

- Census subsection level (subsecção estatística)
- Census tract level
- Small-area census units

Try to obtain Census 2021 data at subsection level.

## BGRI - Base Geográfica de Referenciação da Informação

BGRI provides the geographic boundaries for census units.

INE = attributes
BGRI = geometry

BGRI polygons + INE population data = Population density map

## Porto Open Data

Porto often publishes useful local datasets:

- schools
- parks
- transport
- public facilities
- urban amenities

## OpenStreetMap

Ideal for:

- cafés
- schools
- parks
- libraries
- playgrounds
- transit stops
- street network
